## 파이널 프로젝트 (Lets)

<br />

### 🎤 소개 
Lets는 Let Everyone To Study의 약자로 스터디 카페 예약 서비스와 스터디 그룹 커뮤니티를 제공합니다.

<br />

### 📆 프로젝트 기간
프로젝트 구상 및 DB 설계 : 2023년 08월 28일 ~ 2023년 09월 02일
<br />
화면 및 서버 구현 : 2023년 09월 04일 ~ 2021년 09월 24일
